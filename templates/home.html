Crop Yield Prediction

There are 5 functionalities in Yield Recommender UI page. They are,
• Country Name: User need to select the specific country where he/she wants to check the Yield prediction.
• Rainfall: User needs to enter manually the estimated average rainfall in mm of that area.
• Temperature: User needs to enter manually the estimated temperature in Celsius of that area.
• Pesticides: User needs to enter manually the pesticides usage value that they use in that area.
• Type of Crop: User needs to select the type of crop from the options for he/she to check yield for.
